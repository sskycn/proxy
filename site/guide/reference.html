常见排错速查

  • 启动后没有读到配置：检查当前子命令默认文件名，server 默认 server.json，client 默认 client.json，local/root 默认 config.json。
  • VLESS/VMess 生成失败：确认 --token 是 UUID，或留空让生成器自动生成。
  • WebSocket 连接失败：确认 nginx 路径和 tunnel_path 完全一致，且 Upgrade 头已转发。
  • TLS 失败：确认客户端 tunnel_tls_server_name 与证书域名匹配；测试环境才使用 tls_insecure。
  • REALITY 失败：确认 public key 由同一个 private key 派生，serverName、shortId、flow、UUID 都匹配。
  • local 找不到网关：先显式传 --gateway-ip 和 --gateway-port；再调大 --scan-timeout 检查。
